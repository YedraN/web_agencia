Cursor se ha convertido en el editor de código favorito de los desarrolladores que quieren aprovechar la IA. Basado en VS Code, ofrece todas las funcionalidades del editor de Microsoft potenciadas con IA multimodal.

## ¿Qué hace especial a Cursor?

Cursor integra modelos de lenguaje directamente en el editor. No necesitas copiar y pegar código entre ChatGPT y tu editor. La IA entiende todo tu proyecto, no solo el archivo abierto.

Puedes pedirle a Cursor que modifique funciones, añada tests o refactorice código directamente seleccionando el texto y escribiendo instrucciones en lenguaje natural.

## Funcionalidades clave

**Completado de código**: Sugerencias multi-línea que entienden el contexto de tu proyecto. No solo la línea actual, sino la función completa que intentas escribir.

**Chat inline**: Pregunta sobre el código seleccionado sin cambiar de contexto. "¿Qué hace esta función?", "¿Puedes optimizar este bucle?" o "Añade types a este código".

**Edición multi-archivo**: Cursor puede modificar varios archivos simultáneamente para implementar una funcionalidad completa. Detecta qué archivos necesita cambiar y hace las modificaciones.

**Comandos AI**: Acciones predefinidas como "fix lint errors", "add tests", "write docs" o "explain code" que ejecutan tareas comunes con un solo comando.

## Mejores prácticas

Usa @file y @folder para dar contexto específico. Escribe prompts claros y específicos en lugar de instrucciones vagas. Revisa siempre el código generado antes de aceptarlo.

## Precios

Cursor tiene un plan gratuito limitado, Pro a $20/mes y Business a $40/mes. La versión Pro permite usar GPT-5, Claude 4 y otros modelos ilimitadamente.
