Cada color y tipografía en tu sitio web comunica algo. Los usuarios pueden no notar conscientemente estas elecciones, pero las sienten — y esos sentimientos impulsan decisiones.

## Psicología del color en diseño web

**Azul**: transmite confianza, seguridad y profesionalismo. Es la elección más común para bancos, tecnología y salud.

**Verde**: sugiere crecimiento, salud y conciencia ambiental.

**Rojo**: crea urgencia, emoción o peligro. Efectivo para rebajas y CTAs.

**Púrpura**: comunica creatividad, lujo y sabiduría.

**Naranja**: se siente energético, amigable y asequible.

**Negro y blanco**: comunican sofisticación, minimalismo y claridad.

## Cómo el color afecta las conversiones

El color del botón CTA debe contrastar con el resto de la página. Los tests A/B muestran consistentemente que los botones de alto contraste superan a los de bajo contraste.

## Tipografía y personalidad de marca

**Serif** (Times New Roman, Georgia): se sienten tradicionales, confiables y autoritarios.

**Sans-serif** (Inter, Helvetica, Roboto): se sienten modernos, limpios y accesibles.

**Display**: añaden personalidad para titulares. Úsalos con moderación.

## Mejores prácticas tipográficas

**Legibilidad primero**: texto corporal de al menos 16px. Altura de línea entre 1.5 y 1.7.

**Limita las tipografías**: máximo dos — una para titulares, otra para texto corporal.

**Establece jerarquía**: usa tamaño, peso y color para crear jerarquía visual clara.
