## Introducción

SvelteKit 3 es el framework de aplicaciones web basado en Svelte 5. Con Runes en el núcleo y un nuevo sistema de rutas, es más potente que nunca.

## Nuevo sistema de enrutamiento

SvelteKit 3 introduce rutas basadas en el sistema de archivos con soporte para grupos, layouts anidados y rutas dinámicas mejoradas.

## Form Actions mejoradas

Las acciones de formulario ahora se integran nativamente con Svelte 5 Runes. Los estados de carga y error se manejan con $state de forma limpia.

## Carga de datos optimizada

Los loaders de SvelteKit 3 soportan streaming con `load` asíncrono y caché integrada. Los datos se cargan en paralelo con las rutas.

## Mejoras en el adapter

Los adapters para Node, Vercel, Cloudflare y Netlify reciben actualizaciones que mejoran el rendimiento en edge computing.

## Conclusión

SvelteKit 3 es ideal para aplicaciones que necesitan rendimiento extremo con poco JavaScript. En **Vynta** lo usamos para proyectos que priorizan velocidad de carga.
