OpenAI ha lanzado Canvas, una nueva interfaz que transforma ChatGPT de un chat conversacional a un espacio de trabajo colaborativo. Canvas te permite editar documentos y código en una ventana separada, con cambios en tiempo real.

## ¿Qué es ChatGPT Canvas?

Canvas es un editor lateral que se abre junto al chat tradicional. Cuando ChatGPT genera un documento o fragmento de código, puedes abrirlo en Canvas para editarlo directamente, ver cambios sugeridos y mantener el contexto de la conversación.

## Cómo usarlo para escribir mejor

Con Canvas, la escritura asistida por IA se vuelve mucho más fluida. Puedes seleccionar texto específico y pedir a ChatGPT que lo reformule, acorte o expanda sin regenerar todo el documento.

Las sugerencias de edición aparecen como marcas en el texto, similares al control de cambios de Google Docs, permitiéndote aceptar o rechazar cada modificación.

## Cómo usarlo para programar

Para desarrollo, Canvas es un salto cualitativo. Puedes ver el código completo, no solo fragmentos. Los cambios sugeridos se muestran con diff visual, y puedes depurar líneas específicas sin perder el contexto.

ChatGPT Canvas soporta resaltado de sintaxis para más de 20 lenguajes y ofrece acciones rápidas como "revisar errores", "añadir logs" o "traducir a otro lenguaje".

## Consejos prácticos

Usa Canvas para documentos largos donde necesites mantener consistencia. Para código, es ideal para refactorizar funciones existentes o revisar PRs. La capacidad de hacer cambios inline acelera significativamente la iteración.
