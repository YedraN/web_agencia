## Introduction

Lighthouse 12 arrives with new metrics and stricter audits. Google's tool keeps evolving to evaluate web performance with updated criteria.

## New Metrics

Interaction to Next Paint (INP) becomes an official metric, replacing FID. New metrics for responsiveness and animation smoothness are also added.

## Accessibility Audits

New WCAG 2.2 audits for visible focus, ARIA labels, and color contrast. Lighthouse 12 is stricter on accessibility.

## JavaScript Performance

New audits for evaluating JavaScript's performance impact. Detects heavy libraries, unused code, and render-blocking scripts.

## How to Improve Your Score

Optimize images with modern formats, minimize JavaScript with code splitting, use lazy loading, and ensure fast server response times.
