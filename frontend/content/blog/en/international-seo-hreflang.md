## What is hreflang?

hreflang is an HTML attribute that tells Google the language and geographic region of a page. Essential for multilingual sites.

## Implementation methods

You can use three methods: in the page head with link rel="alternate", in the XML sitemap, or in HTTP headers.

## Common structures

### Country domains (.es, .mx, .com)

Offers the best segmentation but requires more maintenance.

### Subdirectories (/es/, /en/)

Easy to manage and maintains the main domain authority.

### Subdomains (es.site.com)

Useful for very different languages but fragments authority.

## Common mistakes

Not including the current page in the hreflang set, using incorrect language codes, or having inconsistent hreflang across pages.

## Content strategy

Create unique content for each language. Literal translation doesn't work well for SEO. Adapt the message to each culture.
